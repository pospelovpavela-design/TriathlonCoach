На Apple Watch
Как видеть план тренировки прямо на часах
1
iPhone как тренер (уже работает)
Этот экран работает прямо сейчас — добавь страницу на Домашний экран iPhone.

Как добавить: Safari → кнопка «Поделиться» → «На экран Домой». Работает офлайн.
2
WorkoutKit через Команды (iOS 17+)
Приложение «Команды» → «+» → добавить действие:

Ищи «Создать собственную тренировку» (WorkoutKit).
Добавь отрезки с целевым пульсом. Тренировка появится в приложении Тренировка на часах как «Собственная тренировка» с зонами пульса.
3
Ручная настройка зон на часах
Настройки Apple Watch → Тренировка → Целевые зоны:

Введи свои пульсовые зоны вручную. Часы будут вибрировать когда пульс выходит из целевой зоны — это работает с любой тренировкой.
4
Кастомные зоны Павла для Watch
Настройки → Здоровье → Пульсовые зоны → Пользовательские:

Z1: 0–119 уд/мин
Z2: 120–145 уд/мин
Z3: 146–163 уд/мин
Z4: 164–175 уд/мин
Z5: 176+ уд/мин
5
Оптимальный сценарий во время бега
1. Запусти тренировку на часах (бег/вело)
2. На iPhone открой эту страницу — тренер покажет текущий отрезок с таймером
3. Голосовые подсказки объявят переход к следующему отрезку
4. Часы вибрируют при выходе из зоны пульса

Earphones + Apple Watch + iPhone тренер = полная система.
