Часть 4 — Target и Capabilities (самое важное)
8
Что такое Target
Главная концепция которую нужно понять

Target — это «конечный продукт» который Xcode собирает из твоих файлов. Думай об этом как о «профиле сборки»: какие файлы включить, для какого устройства, какие разрешения нужны.

📁 Файлы кода (.swift) + ресурсы
↓ Target собирает всё это в:
📱 TriathlonCoach.app → устанавливается на iPhone

У нас будет один Target — TriathlonCoach. Он же отвечает за синхронизацию с Watch через WorkoutKit.

Если что-то пошло не так
?
Частые ошибки и решения

«No such module 'WorkoutKit'»
→ Вернись к шагу 10, проверь что WorkoutKit добавлен в Frameworks. Проверь что iOS Deployment Target = 17.0+

«Signing certificate is not valid»
→ Xcode → Preferences → Accounts → удали и добавь Apple ID снова

Приложение не устанавливается на iPhone
→ Настройки iPhone → Основные → VPN и управление устройством → найди свой Apple ID → Доверять

Тренировки не видны на Watch
→ Watch должен быть рядом с iPhone и подключён по Bluetooth. Подожди 30 секунд после нажатия кнопки.
